## Хранитель очага

«Хранитель очага» — интерактивная текстовая игра, в которой игрок принимает роль маленького духа, живущего за печкой в старом доме. Однажды в доме начинается пожар, и только Хранитель может спасти жилище от уничтожения. Вам предстоит исследовать комнаты, сражаться с вредителями (мыши, тараканы, пауки, а также огромный кот), решать квесты, собирать полезные предметы и в финале потушить огонь на кухне.

### Технологии
- Twine/SugarCube 2
- JavaScript (ES6)
- HTML5/CSS3
- jQuery

### Игровые механики
- Исследование локаций с вариативностью.
- Визуальное оформление меняется в зависимости от текущей локации.
- Боевая система
    - Пошаговые бои с врагами. 
    - Ходы чередуются: игрок атакует, защищается или использует лечащие предметы, затем ходит враг.
    - Урон зависит от экипированного оружия; без оружия — 1 единица.
    - Защита уменьшает следующий удар врага вдвое.
    - Уклонение — у некоторых врагов есть шанс полностью избежать атаки.
    - Броня поглощает часть урона.
    - Поломка — оружие может сломаться при использовании.
- Инвентарь и предметы
    - Оружие (weapon) — наносит урон, может ломаться.
    - Броня (armor) — уменьшает входящий урон, может ломаться.
    - Расходники (consumable) — восстанавливают HP.
    - Сюжетные (story) — используются для открытия путей или завершения квестов.
    - Экипировка отображается в панели игрока.
    - Инвентарь доступен через кнопку в меню.
- Система квестов
    - Квесты могут быть активными или завершёнными.
    - У каждого квеста есть набор флагов (логических переменных), которые меняются по ходу игры.
    - При выполнении квеста игрок получает награду (предметы, открытие новых локаций).
    - Квесты добавляются в StoryInit и управляются через QuestManager.
- Уведомления
    - Всплывающие уведомления (notify) информируют о получении предметов, начале/завершении квестов, уроне и т.д

### Планируется
- Добавить атмосферные звуки (скрип половиц, шипение огня).
- Доработать финальную сцену.
- Реализовать полноценную систему сохранений.
- Исправить все выявленные баги.
- Добавить «босс-файт».
- Добавить уровень дыма который усложнит прохождение.
